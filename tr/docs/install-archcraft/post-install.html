Yükleme Sonrası

Archcraft'ı yükledikten sonra yapmanız gereken birkaç şey var...

Yükleme / Liveboot sonrası siyah ekran#

Sistem yüklemesinden sonra ya da canlı deneme versiyonunda siyah ekranla karşılaşırsanız, bunun sebebi compositor (Örnek : picom) olabilir. Bu durumda bu sorunu çözmek için :

  • ctrl+alt+f2/f3... kombinasyonlarını kullanarak tty'ye giriş yapınız.
  • ~/.config/picom.conf dosyasını düzenleyin ve backend olarak glx kullanın.
  • Bu işe yaramazsa, compositor picom-ibhagwan-git uygulamasını kaldırın. 
$ sudo pacman -Rsn picom-ibhagwan-git

Sistemi Güncelleme#

Paket veritabanını yenileyin ve yeni kurulumunuzu güncelleyin.

$ sudo pacman -Syyu

Eğer invalid or corrupted package (PGP signature) hatası alırsanız, aşağıdaki komutları uygulayın.

$ sudo pacman -S archlinux-keyring$ sudo pacman-key --populate

Yeni Programlar Yükleme#

pacman ya da yay (AUR) kullanarak favori programlarınızı yüklemek için aşağıdaki komutları kullanın.

$ sudo pacman -S gimp inkscape
# Or from AUR
$ yay -S spotify

Ekran Yırtılmalarını Düzeltme#

Ekran yırtılması sorunu yaşıyorsanız :

  1. Compositor için backend olarak GLX'i etkinleştirin. Eğer openbox kullanıyorsanız masaüstünde sağ tıklayarak Preferences > Compositor yolunu takip edin ve Use 'glx' backend opsiyonunu seçin ya da ~/.config/picom.conf dosyasını manuel olarak düzenleyin.

  2. Eğer ilk seçenek sizin için çalışmadıysa, aşağıdaki adımları takip ederek xorg.conf dosyası oluşturabilirsiniz.

# Change to xorg.conf.d dir$ cd /etc/X11/xorg.conf.d
# Create a conf file$ sudo touch 20-amdgpu.conf
# Edit the file$ sudo vim 20-amdgpu.conf
# Paste this in the editorSection "Device"     Identifier "AMD"     Driver "amdgpu"     Option "TearFree" "true"EndSection
Dikkat

Bu yöntem sisteminizde çalışmayabilir ve xorg'a önyükleme yapamayabilirsiniz. Bu nedenle, yaptığınız değişiklikleri geri almak için tty'yi nasıl kullanacağınızı veya kurtarma moduna nasıl başlayacağınızı bildiğinizden emin olun.

Suspend Servislerini Etkinleştirme#

Kullanıcı hesabınız için betterlockscreen ekran koruyucunun suspend servislerini etkinleştirin.

# Enable betterlockscreen suspend service
$ sudo systemctl enable betterlockscreen@$USER.service

Kilit Ekranı Arka Planını Düzeltme#

Hidpi veya düşük çözünürlüklü ekranlarda kilit ekranı düzenini düzeltin.

# Update the lockscreen wallpaper
$ betterlockscreen -u /usr/share/backgrounds/groot.jpg

Initrd'den Gereksiz Modülleri Kaldırma#

Archcraft'ı (ABIF ile) Intel GPU'lu bir makineye kurduğunuzu varsayalım. Bu durumda sadece i915 modülüne ihtiyacınız var. Modüller listesindeki diğer modülleri silin ve initrd'yi yeniden oluşturun.

# Edit mkinitcpio.conf file$ sudo vim /etc/mkinitcpio.conf
# Change line 7 fromMODULES=(i915? amdgpu? radeon? nouveau? vboxvideo? vmwgfx?)# ToMODULES=(i915)
# Save file and rebuild initrd$ sudo mkinitcpio -p linux

Şimdilik bu kadar. Archcraft'ın tadını çıkarın.
