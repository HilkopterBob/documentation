Problèmes dans la version de septembre

September 30, 2021
Aditya Shakya
Développeur d'Archcraft

Summery#

En raison d'un bug dans les scripts d'installation, les utilisateurs rencontrent quelques problèmes sur l'ISO de septembre. Le support de flatpak a causé des problèmes avec les applications gtk telles que thunar et plank.

Problèmes#

  1. Impossible de régler la luminosité de l'écran
  2. Le lancement de Thunar et de la planche prend beaucoup de temps.
  3. Le greeter de Lightdm se bloque / traîne / affiche une erreur

Solutions#

Vous pouvez essayer ces méthodes pour résoudre les problèmes avec la nouvelle ISO.

Installer les pilotes#

Utilisez lspci pour obtenir les informations sur la ou les cartes graphiques disponibles sur votre système, et installez les pilotes appropriés.

$ lspci -k | grep -P 'VGA|3D|Display' (en anglais)03:00.0 Contrôleur compatible VGA : Advanced Micro Devices, Inc. [AMD/ATI] Picasso (rev c2)

Dans ce cas, installez le paquet xf86-video-amdgpu et redémarrez le système.

Supprimer Flatpak#

Supprimez le paquet flatpak pour résoudre le problème avec les applications gtk.

$ sudo pacman -Rnscv flatpak
caution

Cette opération supprimera également pamac-all car flatpak est l'une de ses dépendances. Vous pouvez installer pamac-aur si vous voulez un Graphical package manager.

Installer Sddm#

En raison de la suppression des pilotes du système installé, le greeter webkit de lightdm se bloque et ne parvient pas à charger le thème par défaut. De plus, il est un peu lent sur les vieilles machines. Pour résoudre ce problème, installez sddm et archcraft-sddm-theme-default. ``bash $ sudo pacman -S sddm archcraft-sddm-theme-default


Copier le fichier de configuration sddm, utiliser **archcraft** comme nom de thème```bash$ sudo cp /usr/lib/sddm/sddm.conf.d/default.conf /etc/sddm.conf.d/default.conf

Activez le service sddm et redémarrez l'ordinateur

$ sudo systemctl enable -f sddm-plymouth.service
info

Tous les problèmes ci-dessus seront corrigés dès que possible et une nouvelle ISO sera bientôt mise en ligne.
